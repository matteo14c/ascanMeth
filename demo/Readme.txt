This folder contains demo data to test the ascanMeth script.
To test the script run Rscript ascanMeth.R
You should see 3 output files:

1. results_D.CpG.csv: results for test of differential density of CpGs between the haplotypes-
2. results_H.CpG.csv: differential methylation of 5mhC
3. results_M_CpG.csv: differential methylation of 5mC
